# 你好！👋

欢迎来到我的技术博客！这是我的第一篇文章，很高兴见到你。

## 关于这个博客

这个博客是使用以下技术构建的：

- **Next.js** - React 框架
- **Tailwind CSS** - 实用优先的 CSS 框架
- **MDX** - Markdown + JSX，让你可以在 Markdown 中使用组件

## 功能特性

这个博客包含了很多现代化的功能：

### 🎨 主题切换
- 支持浅色和深色模式
- 点击右上角的月亮/太阳图标即可切换

### 🔍 搜索功能
- 按下 `Ctrl + K` (Windows) 或 `Cmd + K` (Mac) 打开搜索面板
- 快速搜索所有文章内容

### 💬 评论系统
- 基于 GitHub Discussions 的评论功能
- 可以使用 GitHub 账号直接评论

### 📱 响应式设计
- 完美适配手机、平板和桌面设备
- 在任何设备上都有良好的阅读体验
